# Employee Management System

## Overview
This project is a Java console-based Employee Management System that uses SQLite for database storage.

## Features
- Add employees
- View employees
- Update employee information
- Delete employees
- SQLite database integration

## Technologies Used
- Java
- SQLite
- JDBC

## OOP Concepts Demonstrated
- Abstraction
- Inheritance
- Polymorphism
- Interfaces
- Composition
- Constructors
- Access Specifiers

## Project Summary
This project demonstrates how object-oriented programming concepts and database interactions can be combined to create a functional employee management system.
